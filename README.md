# Movie World 🎬

## About the Project

Movie World is a web application that helps users discover movies in a simple and interactive way. Users can search for movies, explore different genres, and find something new to watch without wasting time scrolling.

One of the key features of this app is a random movie suggestion system. Users first select a genre, and then the app suggests a random movie from that category, making discovery faster and more engaging.

---

## Purpose

This project is built to demonstrate:

* API integration using fetch
* Working with dynamic data in JavaScript
* Use of array methods like map, filter, and sort
* Building a responsive and clean user interface

---

## API Used

TMDB (The Movie Database) API

The API provides:

* Movie titles
* Posters
* Ratings
* Release dates
* Genres and descriptions

---

## Features

### Core Features

* Search movies by name
* Browse trending or popular movies
* Filter movies by genre
* Sort movies by rating or release date
* View detailed information about each movie

### Random Movie Suggestion

* User selects a genre
* On clicking the button, a random movie from that genre is suggested
* Helps users quickly decide what to watch

### Favorites

* Add or remove movies from favorites
* Data stored using localStorage

## How It Works

The application fetches movie data from the TMDB API and displays it dynamically. Users can search, filter, and sort movies using different options. The random suggestion feature selects a movie based on the chosen genre. Array higher-order functions are used to handle and manipulate data efficiently.
